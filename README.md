# KanBan3D

A 3D neumorphic Kanban board demo built with React + Vite.

## Features

- **3D Neumorphic UI** — Realistic raised/pressed button and card effects
- **Drag & Drop** — Move tasks between columns with visual feedback
- **Dark Theme** — Modern dark mode with purple accent colors
- **Accessibility** — Respects `prefers-reduced-motion`

## Tech Stack

- React 19
- Vite
- CSS Custom Properties (design tokens)
- Native HTML5 Drag and Drop API

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
